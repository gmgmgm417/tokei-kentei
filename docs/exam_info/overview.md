# 統計検定準1級 試験概要

## 試験の基本情報

| 項目 | 内容 |
|---|---|
| 試験名 | 統計検定 準1級 |
| 主催 | 一般財団法人 統計質保証推進協会 |
| 試験形式 | CBT（Computer Based Testing） |
| 試験時間 | 90分 |
| 問題数 | 40問（選択式） |
| 合格基準 | 概ね65%以上（70点/100点程度） |
| 受験料 | 7,000円（税込） |

## 出題分野と出題割合（目安）

| 分野 | 出題割合 | 重要度 |
|---|---|---|
| 統計的推測（推定・検定） | 約25% | ★★★★★ |
| 確率論・確率分布 | 約20% | ★★★★★ |
| 回帰分析 | 約15% | ★★★★☆ |
| 線形モデル（ANOVA） | 約15% | ★★★★☆ |
| 多変量解析 | 約10% | ★★★★☆ |
| ベイズ統計 | 約8% | ★★★☆☆ |
| 時系列解析 | 約7% | ★★★☆☆ |

!!! warning "注意"
    上記は目安です。出題割合は試験回によって変動します。

## 試験の特徴

!!! info "出題傾向"
    - **計算問題が多い**: 数値を計算させる問題が全体の50〜60%
    - **概念理解問題**: 定義や性質を正しく理解しているか
    - **数式の選択**: 正しい数式・手順を選ぶ
    - **解釈問題**: 統計出力や結果の正しい解釈

!!! tip "高得点のポイント"
    1. 公式は暗記より導出を理解する
    2. 計算問題は確実に得点する
    3. 時間配分（90分で40問 = 1問約2分）
    4. 分布の性質（期待値・分散）を素早く答えられるようにする
