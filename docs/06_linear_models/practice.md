# 線形モデル：練習問題

## 予想問題 L-001：平方和の分解

**難易度**: ★★★☆☆
**頻出度**: 高

3グループのデータがある。

| | 観測値 | 平均 |
|---|---|---|
| グループ1 | 2, 4, 6 | 4 |
| グループ2 | 3, 5, 7 | 5 |
| グループ3 | 6, 8, 10 | 8 |
| **全体平均** | | **17/3 ≈ 5.67** |

$SS_A$（群間）と $SS_E$（群内）を計算せよ。

??? success "解答・解説"

    **全体平均**: $(4+5+8)/3 = 17/3$

    **$SS_A$（群間平方和）**:

    $$SS_A = \sum_i n_i(\bar{Y}_{i\cdot} - \bar{Y})^2 = 3\left[(4-\frac{17}{3})^2 + (5-\frac{17}{3})^2 + (8-\frac{17}{3})^2\right]$$

    $$= 3\left[\frac{25}{9} + \frac{4}{9} + \frac{49}{9}\right] = 3 \times \frac{78}{9} = 26$$

    **$SS_E$（群内平方和）**:
    - G1: $(2-4)^2 + (4-4)^2 + (6-4)^2 = 4+0+4 = 8$
    - G2: $(3-5)^2 + (5-5)^2 + (7-5)^2 = 4+0+4 = 8$
    - G3: $(6-8)^2 + (8-8)^2 + (10-8)^2 = 4+0+4 = 8$
    - $SS_E = 8+8+8 = 24$

## 予想問題 L-002：多重比較の選択

**難易度**: ★★☆☆☆
**頻出度**: 中

以下の状況で適切な多重比較法を選べ。

(1) 5つの新薬群とコントロール群（プラセボ）の比較
(2) 4つの処理群の全ての対比較

??? success "解答・解説"

    **(1) ダネット法**

    コントロール群との比較のみが目的 → ダネット法が最も統計的な検出力が高い

    **(2) テューキー法**

    全ての対比較（$\binom{4}{2}=6$ 通り）→ テューキー法（Tukey's HSD）が適切
