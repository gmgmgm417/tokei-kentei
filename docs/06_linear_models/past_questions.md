# 線形モデル：過去問

## 問題 L1-001 (基礎) ：一元配置ANOVAの自由度

> **出典**: 統計検定準1級 類題

3つの群（$a=3$）、各群10個（$n_i=10$）のデータがある一元配置ANOVAのANOVA表を完成せよ。
$SS_A = 120$、$SS_E = 270$

??? success "解答・解説を見る"

    | 変動要因 | 平方和 | 自由度 | 平均平方 | $F$ 値 |
    |---|---|---|---|---|
    | 群間 | 120 | 2 | 60 | 6.0 |
    | 群内 | 270 | 27 | 10 | |
    | 全体 | 390 | 29 | | |

    - 群間自由度: $a-1 = 3-1 = 2$
    - 群内自由度: $N-a = 30-3 = 27$
    - $MS_A = 120/2 = 60$、$MS_E = 270/27 = 10$
    - $F = MS_A/MS_E = 60/10 = 6.0$

    !!! tip "有意性の判断"
        $F_{0.05}(2, 27) \approx 3.35$ なので $F=6.0 > 3.35$ で有意差あり。

## 問題 L1-002 (応用) ：交互作用の解釈

> **出典**: 統計検定準1級 類題

二元配置ANOVAで交互作用が有意だった場合、主効果の解釈はどうすべきか。

??? success "解答・解説を見る"

    **主効果の単純な解釈は不適切。**

    交互作用が有意 = 因子Aの効果の大きさ（または方向）が因子Bの水準によって異なることを意味する。

    この場合:
    - 「因子Aに全体的な効果がある」と単純に言えない
    - 因子Bの各水準ごとに因子Aの効果を検討する「単純主効果の検定」が必要
    - プロファイルプロット（交互作用図）で視覚的に確認する

    !!! example "交互作用の例"
        薬品A・Bの効果が、男性では「AがBより効く」が、女性では「BがAより効く」場合 → 性別と薬品に交互作用がある
