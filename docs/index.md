## 使い方

!!! tip "スマホでの学習方法"
    - **ホーム画面に追加**してアプリのように使う（Safari/Chrome対応）
    - 右上の **:material-magnify: 検索** で公式・概念をすぐに検索
    - 問題は **タップで解答を展開** できる折りたたみ式
    - **ダークモード** の切り替えで夜間・日中どちらでも快適に

!!! note "コンテンツフォーマット"
    各分野のページは以下の構成です：

    - **概要** → トピックの全体像と重要度
    - **重要概念・公式** → 試験に出る公式と解説
    - **過去問** → 折りたたみ式の問題・解答
    - **練習問題** → 予想問題・ヒント付き

---

## 学習ロードマップ

```
1. 確率論（基礎固め）
   ↓
2. 統計的推測（最重要・最多出題）
   ↓
3. 回帰分析（計算問題を練習）
   ↓
4. 線形モデル（ANOVA）
   ↓
5. 多変量解析（PCAを重点的に）
   ↓
6. ベイズ統計
   ↓
7. 時系列解析
```
