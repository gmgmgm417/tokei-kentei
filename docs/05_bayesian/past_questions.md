# ベイズ統計：過去問

## 問題 B1-001 (応用) ：共役事前分布

> **出典**: 統計検定準1級 類題

コインを $n=20$ 回投げて表が $x=12$ 回出た。表の確率 $\theta$ の事前分布を $Be(2, 2)$ とするとき：
(1) 事後分布を求めよ。
(2) $\theta$ のベイズ推定量（事後期待値）を求めよ。

??? success "解答・解説を見る"

    **(1)** $\theta \mid x \sim Be(14, 10)$

    二項尤度に対するベータ事前分布の更新：
    $$Be(\alpha, \beta) \to Be(\alpha + x, \beta + n - x) = Be(2+12, 2+8) = Be(14, 10)$$

    **(2)** ベイズ推定量 $= \frac{14}{14+10} = \frac{14}{24} = \frac{7}{12} \approx 0.583$

    ベータ分布 $Be(\alpha, \beta)$ の期待値は $\frac{\alpha}{\alpha+\beta}$。

    !!! info "比較"
        - MLE: $\hat{\theta}_{MLE} = 12/20 = 0.6$
        - ベイズ推定量: $7/12 \approx 0.583$
        - 事前期待値: $2/(2+2) = 0.5$

        ベイズ推定量は MLE と事前期待値の間に位置する（サンプル数が増えると MLE に近づく）。

## 問題 B1-002 (基礎) ：信用区間 vs 信頼区間

> **出典**: 統計検定準1級 類題

95%信用区間と95%信頼区間の解釈の違いを説明せよ。

??? success "解答・解説を見る"

    **95%信頼区間**: 同じ手続きを繰り返したとき、95%の割合でパラメータを含む区間を作れる。特定の実現した区間がパラメータを含む確率は 0 か 1 のどちらかである。

    **95%信用区間**: データを観測した後、パラメータがその区間内にある（事後）確率が95%。より直感的な解釈が可能。

    !!! warning "よくある誤解"
        「95%信頼区間にパラメータが含まれる確率は95%」は厳密には誤り（頻度論の枠組みでは）。
        ただし実用上はこのように解釈されることも多い。
