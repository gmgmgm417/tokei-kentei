## 他分野との関連

確率論は以下の全分野の基礎です：

- **統計的推測** → 確率分布の理解が不可欠
- **ベイズ統計** → 事前・事後分布の計算
- **多変量解析** → 多変量正規分布
- **時系列解析** → 確率過程の基礎
