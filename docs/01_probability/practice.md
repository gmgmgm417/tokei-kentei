## 予想問題 P-001：条件付き確率

**難易度**: ★★☆☆☆
**頻出度**: 高

コインを3回投げる。少なくとも1回表が出たという条件のもとで、ちょうど1回だけ表が出る確率を求めよ。

??? hint "ヒント"
    条件付き確率の定義 $P(A \mid B) = P(A \cap B)/P(B)$ を使う。

    $A$: ちょうど1回表、$B$: 少なくとも1回表 として計算する。

??? success "解答・解説"

    **正解**: $3/7$

    ### 計算過程

    コインを3回投げる試行では全8通り（$2^3 = 8$）の等確率の結果がある。

    - $P(B) = P(\text{少なくとも1回表}) = 1 - P(\text{全て裏}) = 1 - 1/8 = 7/8$
    - $P(A \cap B) = P(\text{ちょうど1回表}) = \binom{3}{1}(1/2)^3 = 3/8$

    $$P(A \mid B) = \frac{P(A \cap B)}{P(B)} = \frac{3/8}{7/8} = \frac{3}{7}$$

## 予想問題 P-002：分布関数と密度関数

**難易度**: ★★★☆☆
**頻出度**: 中

確率変数 $X$ の確率密度関数が
$$f(x) = \begin{cases} cx^2 & (0 < x < 2) \\ 0 & (\text{otherwise}) \end{cases}$$
であるとき、(1) 定数 $c$ を求めよ。(2) $P(1 \leq X \leq 2)$ を求めよ。

??? hint "ヒント"
    密度関数の条件: $\int_{-\infty}^{\infty} f(x) \, dx = 1$

??? success "解答・解説"

    **(1)** $c = 3/8$

    $$\int_0^2 cx^2 \, dx = c \cdot \left[\frac{x^3}{3}\right]_0^2 = c \cdot \frac{8}{3} = 1$$
    $$\therefore c = \frac{3}{8}$$

    **(2)** $7/8$

    $$P(1 \leq X \leq 2) = \int_1^2 \frac{3}{8}x^2 \, dx = \frac{3}{8}\left[\frac{x^3}{3}\right]_1^2 = \frac{3}{8} \cdot \frac{8-1}{3} = \frac{7}{8}$$

## 予想問題 P-003：独立性の判定

**難易度**: ★★★☆☆
**頻出度**: 中

二つの確率変数 $X, Y$ の同時確率関数が以下の表で与えられるとき、$X$ と $Y$ は独立かどうか判定せよ。

| | $Y=0$ | $Y=1$ |
|---|---|---|
| $X=0$ | $1/4$ | $1/4$ |
| $X=1$ | $1/4$ | $1/4$ |

??? hint "ヒント"
    独立の定義: $P(X=x, Y=y) = P(X=x) \cdot P(Y=y)$ が全ての $x, y$ で成立。

??? success "解答・解説"

    **独立である**

    周辺確率を計算する：
    - $P(X=0) = 1/4 + 1/4 = 1/2$
    - $P(X=1) = 1/4 + 1/4 = 1/2$
    - $P(Y=0) = 1/4 + 1/4 = 1/2$
    - $P(Y=1) = 1/4 + 1/4 = 1/2$

    全ての組合せで確認：
    $$P(X=0, Y=0) = 1/4 = (1/2)(1/2) = P(X=0)P(Y=0) \checkmark$$

    全組合せで $P(X=x, Y=y) = P(X=x)P(Y=y)$ が成立するので、$X$ と $Y$ は独立。

## 予想問題 P-004：正規分布の標準化

**難易度**: ★★☆☆☆
**頻出度**: 高

$X \sim N(50, 100)$ のとき、$P(40 \leq X \leq 65)$ を標準正規分布の分布関数 $\Phi$ で表せ。

??? hint "ヒント"
    標準化: $Z = (X - \mu)/\sigma$

??? success "解答・解説"

    **正解**: $\Phi(1.5) - \Phi(-1)$

    $\mu = 50$、$\sigma^2 = 100$ より $\sigma = 10$。

    標準化すると：
    $$P(40 \leq X \leq 65) = P\!\left(\frac{40-50}{10} \leq Z \leq \frac{65-50}{10}\right) = P(-1 \leq Z \leq 1.5)$$
    $$= \Phi(1.5) - \Phi(-1)$$

    !!! tip "数値計算"
        $\Phi(1.5) \approx 0.9332$、$\Phi(-1) = 1 - \Phi(1) \approx 1 - 0.8413 = 0.1587$

        よって $P \approx 0.9332 - 0.1587 = 0.7745$
