# 時系列解析：練習問題

## 予想問題 T-001：ARIMAモデルの識別

**難易度**: ★★★☆☆
**頻出度**: 高

以下の時系列データの特徴を持つとき、適切なARIMAモデルを選べ。

- 元のデータのACFはゆっくりと減衰する
- 1階差分後のデータのACFはラグ1以降ほぼゼロ
- 1階差分後のデータのPACFは指数的に減衰

??? success "解答・解説"

    **正解**: ARIMA(0,1,1)（= IMA(1,1)）

    ### 判断プロセス

    1. 元のACFがゆっくり減衰 → **非定常**（単位根あり）
    2. 1階差分で定常化 → $d = 1$
    3. 差分後: ACFがラグ1以降ゼロ → MA(1)の特徴
    4. 差分後: PACFが指数減衰 → MA過程の特徴

    よって ARIMA(0, 1, 1)。

## 予想問題 T-002：AR(1)の分散

**難易度**: ★★★☆☆
**頻出度**: 中

$X_t = 0.8 X_{t-1} + \varepsilon_t$（$\varepsilon_t \sim N(0, 1)$）のとき、$\text{Var}(X_t)$ を求めよ。

??? success "解答・解説"

    **正解**: $\text{Var}(X_t) = \frac{\sigma^2}{1-\phi^2} = \frac{1}{1-0.64} = \frac{1}{0.36} \approx 2.78$

    ### 計算過程

    定常AR(1)の分散公式:
    $$\text{Var}(X_t) = \phi^2\text{Var}(X_t) + \sigma^2$$
    $$\text{Var}(X_t)(1-\phi^2) = \sigma^2$$
    $$\text{Var}(X_t) = \frac{\sigma^2}{1-\phi^2} = \frac{1}{1-0.64} = \frac{1}{0.36} = \frac{25}{9}$$
