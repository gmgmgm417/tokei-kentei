# 時系列解析：過去問

## 問題 T1-001 (基礎) ：ACF・PACFの読み方

> **出典**: 統計検定準1級 類題

あるモデルのACFがラグ2以降でほぼゼロになり、PACFは指数的に減衰している。
このデータに最も適したモデルはどれか。

1. AR(2)
2. MA(2)
3. ARMA(1,1)
4. AR(1)

??? success "解答・解説を見る"

    **正解**: 2（MA(2)）

    ACFがラグ $q$ 以降でカットオフ → MA($q$) の特徴
    PACFが指数減衰 → MA過程の特徴

    ラグ2以降でACFがゼロ → MA(2)が最適

    !!! tip "ACF/PACFの識別表"
        | ACFの形 | PACFの形 | 推奨モデル |
        |---|---|---|
        | ラグ$q$以降ゼロ | 減衰振動・指数減衰 | MA($q$) |
        | 減衰振動・指数減衰 | ラグ$p$以降ゼロ | AR($p$) |
        | 減衰振動・指数減衰 | 減衰振動・指数減衰 | ARMA($p$,$q$) |

## 問題 T1-002 (基礎) ：AR(1)の定常性

> **出典**: 統計検定準1級 類題

AR(1)モデル $X_t = \phi X_{t-1} + \varepsilon_t$ が定常となるための必要十分条件を述べよ。

??? success "解答・解説を見る"

    **正解**: $|\phi| < 1$

    特性方程式: $1 - \phi z = 0$ の根 $z = 1/\phi$ が単位円の外側、すなわち $|1/\phi| > 1$、つまり $|\phi| < 1$。

    !!! info "定常AR(1)の性質"
        - $E[X_t] = 0$（$\mu=0$の場合）
        - $\text{Var}(X_t) = \sigma^2/(1-\phi^2)$
        - $\rho(h) = \phi^h$（ACFが指数減衰）
