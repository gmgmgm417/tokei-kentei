    確率論：過去問¶

    利用方法

    問題を読んで解答を考えてから、「解答・解説を見る」をタップしてください。 折りたたみ式になっているので、スマホでの学習に最適です。

    問題 P1-001 (基礎) ：ベイズの定理¶

    出典: 統計検定準1級 類題

    ある疾患の有病率は1%である。この疾患の検査は、有病者の90%が陽性、健常者の5%が陽性となる。 検査で陽性だった人がこの疾患に罹患している確率を求めよ。

    解答・解説を見る

    正解: 約15.3%

    解法の方針¶

    ベイズの定理を使って、\(P(\text{疾患} \mid \text{陽性})\) を求める。

    計算過程¶

    事象を定義する： - \(D\): 疾患あり、\(D^c\): 疾患なし - \(T\): 検査陽性

    与えられた情報： - \(P(D) = 0.01\)、\(P(D^c) = 0.99\) - \(P(T \mid D) = 0.90\)（感度） - \(P(T \mid D^c) = 0.05\)（偽陽性率）

    全確率の定理で分母を計算： $\(P(T) = P(T \mid D)P(D) + P(T \mid D^c)P(D^c)\)$

    \[= 0.90 \times 0.01 + 0.05 \times 0.99 = 0.009 + 0.0495 = 0.0585\]

    ベイズの定理を適用： $\(P(D \mid T) = \frac{P(T \mid D)P(D)}{P(T)} = \frac{0.90 \times 0.01}{0.0585} = \frac{0.009}{0.0585} \approx 0.1538\)$

    ポイント¶

    試験対策

    有病率が低い場合、高精度な検査でも陽性的中率は低くなる（ベースレートの無視）。 この性質は統計検定でよく問われる。

    よくある間違い

    「感度90%だから陽性なら90%の確率で疾患あり」というのは誤り。 事前確率（有病率）を考慮する必要がある。

    問題 P1-002 (基礎) ：期待値の計算¶

    出典: 統計検定準1級 類題

    \(X \sim N(\mu, \sigma^2)\) のとき、\(E[X^2]\) を求めよ。

    解答・解説を見る

    正解: \(\mu^2 + \sigma^2\)

    解法の方針¶

    分散の定義式を利用する。

    計算過程¶

    分散の定義より： $\(\text{Var}(X) = E[X^2] - (E[X])^2\)$

    よって： $\(E[X^2] = \text{Var}(X) + (E[X])^2 = \sigma^2 + \mu^2\)$

    ポイント¶

    試験対策

    \(E[X^2] = \text{Var}(X) + (E[X])^2\) は頻出変形。 この形は多くの計算問題で使える。

    問題 P1-003 (応用) ：モーメント母関数¶

    出典: 統計検定準1級 類題

    \(X_1, X_2, \ldots, X_n\) が \(i.i.d. \sim N(\mu, \sigma^2)\) のとき、\(\bar{X} = \frac{1}{n}\sum_{i=1}^n X_i\) の分布を求めよ。

    解答・解説を見る

    正解: \(\bar{X} \sim N\!\left(\mu, \frac{\sigma^2}{n}\right)\)

    解法の方針¶

    モーメント母関数（MGF）の積の性質を使う。

    計算過程¶

    \(X_i \sim N(\mu, \sigma^2)\) の MGF は： $\(M_{X_i}(t) = \exp\!\left(\mu t + \frac{\sigma^2 t^2}{2}\right)\)$

    \(\sum X_i\) の MGF（独立性より積）： $\(M_{\sum X_i}(t) = \prod_{i=1}^n M_{X_i}(t) = \exp\!\left(n\mu t + \frac{n\sigma^2 t^2}{2}\right)\)$

    \(\bar{X} = \frac{1}{n}\sum X_i\) の MGF（\(at\) を代入）： $\(M_{\bar{X}}(t) = M_{\sum X_i}\!\left(\frac{t}{n}\right) = \exp\!\left(\mu t + \frac{\sigma^2 t^2}{2n}\right)\)$

    これは \(N(\mu, \sigma^2/n)\) の MGF と一致するので、\(\bar{X} \sim N(\mu, \sigma^2/n)\)。

    試験対策

    正規分布の再生性はMGFで示す。 「標本平均の分散は \(\sigma^2/n\)」は基本事項として即答できるようにする。

    問題 P1-004 (基礎) ：チェビシェフの不等式¶

    出典: 統計検定準1級 類題

    \(E[X] = 5\)、\(\text{Var}(X) = 4\) のとき、\(P(|X - 5| \geq 4)\) の上界をチェビシェフの不等式を使って求めよ。

    解答・解説を見る

    正解: \(1/4\)

    計算過程¶

    チェビシェフの不等式： $\(P(|X - \mu| \geq k\sigma) \leq \frac{1}{k^2}\)$

    ここで \(\mu = 5\)、\(\sigma^2 = 4\) なので \(\sigma = 2\)。 \(k\sigma = 4\) より \(k = 4/2 = 2\)。

    \[P(|X - 5| \geq 4) \leq \frac{1}{2^2} = \frac{1}{4}\]

    試験対策

    \(k = \frac{\text{偏差量}}{\sigma}\) として計算する。 チェビシェフは分布の仮定なしで使える（任意の分布に対して成立）。

    問題 P1-005 (応用) ：3事象の確率・加法定理¶

    出典: 統計検定準1級 類題（表現を改変）

    3つの事象 \(A, B, C\) について以下の確率が与えられている：

    \[P(A) = 0.45, \quad P(A \cup B) = 0.65, \quad P(A \mid B) = 0.5\]
    \[P(C) = 0.45, \quad P(A \cap C) = 0.2, \quad P(B \cap C) = 0.1, \quad P(A \cap B \cap C) = 0.05\]

    (1) 最初の3つの条件から \(P(B)\) を求めよ。

    (2) \(P(A \cup B \cup C)\) を求めよ。

    解答・解説を見る

    (1) \(P(B) = 0.40\)

    計算過程¶

    条件付き確率の定義より： $\(P(A \mid B) = \frac{P(A \cap B)}{P(B)} = 0.5 \implies P(A \cap B) = 0.5 \cdot P(B)\)$

    加法定理に代入： $\(P(A \cup B) = P(A) + P(B) - P(A \cap B)\)$

    \[0.65 = 0.45 + P(B) - 0.5 \cdot P(B) = 0.45 + 0.5 \cdot P(B)\]
    \[P(B) = \frac{0.20}{0.5} = 0.40\]

    (2) \(P(A \cup B \cup C) = 0.85\)

    計算過程¶

    (1) より \(P(A \cap B) = 0.5 \times 0.40 = 0.20\)

    3事象の加法定理： $\(P(A \cup B \cup C) = P(A) + P(B) + P(C) - P(A \cap B) - P(A \cap C) - P(B \cap C) + P(A \cap B \cap C)\)$

    \[= 0.45 + 0.40 + 0.45 - 0.20 - 0.20 - 0.10 + 0.05 = \mathbf{0.85}\]

    試験対策

    3事象の加法定理は符号に注意。\(-P(\cdot \cap \cdot)\) が3項、\(+P(\cdot \cap \cdot \cap \cdot)\) が1項。

    よくある間違い

    \(P(A \cap B)\) を最初に求めてから代入する2段階の手順を忘れずに。
